Dev snapshot: Godot 4.1 dev 2

By: Rémi Verschelde
10 May 2023
Pre-release

The development phase for Godot 4.1 is well under way, still aiming for the stable release by the end of June / early July. We had a first dev snapshot a few weeks ago, and it’s now time for the second one.

This build includes a number of big PRs which you might want to test specifically:

  • 2D: Add proper snapping to tile polygon editor (GH-70488).
  • 3D: Fixes to CSG robustness (GH-74771).
  • Core: Prevent errors when using ViewportTexture (GH-75751).
  • GDScript: Improve GDScript documentation generation & behavior (GH-72095).
  • GDScript: Add support for static variables in GDScript (GH-76264).
  • GUI: Add support for multiline cells to Tree (GH-61714).
  • Network: Redo how the remote filesystem works (GH-76540).
  • Porting: Android: Allow concurrent buffering and dispatch of input events (GH-76399).
  • Rendering: Fix voxel GI issues (GH-76437, GH-76550).
  • Rendering: Add NoiseTexture3D (GH-76486).
  • Shaders: Add shader cache to GLES3 (GH-76092).

A lot of work is also being done to improve the Android editor port. We just created a separate Play Store release for the Godot 4.1 dev snapshots, so that interested users can test it easily and provide us with feedback and automated reports on potential issues. You can join the testing group here to get access.

Jump to the Downloads section.

You can also try the Web editor.

The illustration picture for this article is from Spring Dash, a momentum-packed parkour platformer by Dillon Steyl, which was just ported to Godot 4. You can wishlist it on Steam, and find a demo on Dillon’s Discord (find all links here).

What’s new

We now have a great interactive changelog you can use to review all 150 or so changes since the previous dev snapshot more extensively, with convenient links to the relevant PRs on GitHub.

Here are some of the main changes you might be interested in:

  • 2D: Add proper snapping to tile polygon editor (GH-70488).
  • 2D: Tilemaps: Add method to fetch the layer for a given body (GH-76246).
  • 2D: Improve reliability of 2D shape editor redrawing (GH-76492).
  • 3D: Fixes to CSG robustness (GH-74771).
  • 3D: Fix infinite loop in CSG Build2DFaces::_find_edge_intersections (GH-76521).
  • 3D: Fix SurfaceTool::create_from_blend_shape() (GH-76669).
  • Animation: Expose interpolation methods for 3D track in Animation class (GH-73656).
  • Core: Use String.repeat() to optimize several String methods (GH-72288).
  • Core: Add --quit-after <number-of-iterations> (GH-73617).
  • Core: Reimplement String.erase() as immutable method (GH-75510).
  • Core: Prevent errors when using ViewportTexture (GH-75751).
  • Core: Expose determinant in Transform2D, rename internal method (GH-76311).
  • Core: Fix thread IDs (GH-76345).
  • Core: Improve and document PackedDataContainer (GH-76561).
  • Core: Support long path in file access on Windows (GH-76739).
  • Core: Cache feature list in OS.has_feature() (GH-76748).
  • Core: Add ValidatedCall to MethodBind (GH-76418).
  • Editor: Add editor setting for spin slider sensibility (GH-50671).
  • Editor: Prompt to confirm anim track delete on node delete (GH-58598).
  • Editor: Android: Fix UI responsiveness to touch taps (GH-75703).
  • Editor: Android: Adds a scale_gizmo_handles entry to the Touchscreen editor settings (GH-75718).
  • Editor: Make create folder popup support nested folders (GH-76084).
  • Editor: Command Palette search now also uses original English command names (GH-76523).
  • Editor: Preserve scene unique names when saving branch as scene (GH-76609).
  • Export: Explicitly mark inherited export mode when making a dedicated server export (GH-76700).
  • GDExtension: Add handling of custom visual shader nodes from GDExtension (GH-70911).
  • GDScript: Improve GDScript documentation generation & behavior (GH-72095).
  • GDScript: Add support for static variables in GDScript (GH-76264).
  • GDScript: Reorganize and unify warnings (GH-76412).
  • GDScript: Don’t fail when freed object is returned (GH-76483).
  • GUI: Add support for multiline cells to Tree (GH-61714).
  • GUI: Add center_grabber theme property to Slider (GH-69053).
  • GUI: Add icon_modulate set/get functionality to PopupMenu (GH-70286).
  • GUI: Expose horizontal/vertical custom_step as editor property for the ScrollContainer (GH-70868).
  • GUI: Implement vertical icon alignment for buttons (GH-74369).
  • GUI: Add an option for ButtonGroups to be unpressed (GH-76279).
  • GUI: Add more uses of appropriate cursors when resizing/moving some UI nodes (GH-76809).
  • Import: Fix Silhouette used incorrect index (GH-76499).
  • Import: Use DXT1 when compressing PNGs with RGB format (GH-76516).
  • Network: Redo how the remote filesystem works (GH-76540).
  • Physics: Propagate previously unused NOTIFICATION_WORLD_2D_CHANGED, make CanvasItem/CollisionObject2D use it (GH-57179).
  • Physics: Fix precision in physics supports generation (GH-76379).
  • Physics: Add debug collision shape to CSG with collision (GH-76675).
  • Porting: Implement and expose OS.shell_show_in_file_manager() (GH-69698, GH-76428).
  • Porting: Implement and expose to scripting APIs get_memory_info method instead of old get_free_static_memory (GH-75640).
  • Porting: Android: Downgrade android gradle plugin to version 7.2.1 (GH-76325).
  • Porting: Android: Allow concurrent buffering and dispatch of input events (GH-76399).
  • Porting: Android: Fix dynamic Variant params stack constructions in JNI callbacks (GH-76640).
  • Porting: Android: Fix double tap & drag on Android (GH-76791).
  • Porting: iOS: Fix loading of GDExtension dylibs auto converted to framework (GH-76510).
  • Porting: macOS: Bump min. version to 10.13, and remove deprecated code (GH-76394).
  • Rendering: Fix GLES3 rendering on Android studio emulator (GH-74945).
  • Rendering: Add LIGHT_IS_DIRECTIONAL built-in for spatial shaders (GH-76290).
  • Rendering: Fix issues with Vulkan layout transitions (GH-76315).
  • Rendering: Fix voxel GI issues (GH-76437, GH-76550).
  • Rendering: Add NoiseTexture3D (GH-76486).
  • Rendering: Use proper UV in cubemap downsampler raster (Fixes reflections in mobile renderer) (GH-76692).
  • Shaders: Add shader cache to GLES3 (GH-76092).

Bug reports

As a tester, we encourage you to open bug reports if you experience issues with this release. Please check the existing issues on GitHub first, using the search function with relevant keywords, to ensure that the bug you experience is not already known.

In particular, any change that would cause a regression in your projects is very important to report (e.g. if something that worked fine in 4.0.x, but no longer works in 4.1 dev 2).
